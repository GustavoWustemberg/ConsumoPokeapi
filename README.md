<p>This project was created to train my knowledge in python dealing with data from an external api and using (OOP) object-oriented programming.</p>
<h2>Running</h2>
<ul>
    <li>Run `pip install -r requirements.txt` to install the packages</li>
    <li>Run `py app,py`</li>
</ul>
